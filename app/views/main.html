CRUD Kit

CRUD Kit helps you structure and manage your models, forms, fields, validations and api calls. All the boring stuff.

Define your models using JSON Schemas

Define your view models using JSON schemas. This can be done via angular.module.config() and crudKitConfigProvider.addSchema(). See app/scripts/demo.js for an example.

Build forms using these definitions

Customizable form builder with validation and ui-bootstrap integration.

Using the ckForm and ckField directives, you can quickly build forms that automatically adds validation and chooses the correct input template. Just specify the schema name, model instance, and onSave/Delete actions.
